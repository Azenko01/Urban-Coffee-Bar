# ☕ Urban Coffee Bar Website

Urban Coffee Bar is a stylish website concept for a modern café or coffee shop.  
The project focuses on presenting coffee drinks, menu items, and creating a welcoming digital experience that reflects the cozy atmosphere of an urban café.

## 🚀 Live Demo


## 📌 Features
- Modern coffee shop landing page
- Coffee and drinks menu showcase
- Clean and minimal layout
- Responsive design for all devices
- Call-to-action sections for visitors
- Visually appealing design for food & beverage businesses

## 🛠️ Tech Stack
- TypeScript
- CSS
- JavaScript

## 🎯 Project Purpose
The goal of this project was to create a visually appealing front-end website for cafés and coffee shops while improving layout design and responsive development skills.
